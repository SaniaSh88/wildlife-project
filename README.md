# Wildlife Conservation Platform

A C++ application designed to manage a comprehensive wildlife catalog. This system allows users to track animal species, manage global habitats, and organize professional wildlife photography data.

## 🚀 Features
* **Wildlife Database:** Track, view, and organize diverse animal species.
* **Habitat Management:** Monitor ecological zones and conservation environments.
* **Photography Gallery:** Map multimedia photos directly to their corresponding catalog entries.
* **Statistics Dashboard:** Generate automated breakdowns of conservation data.

## 🛠️ Technology Stack
* **Language:** C++ (Object-Oriented Programming)
* **Build System:** Makefile / GCC Compiler
* **Data Structure:** Built using Composition (Catalog contains Species, Habitats, and Photos)

## 💻 How to Run
1. Open your terminal in the project directory.
2. Compile and launch using the makefile:
   ```bash
   mingw32-make run
   ```
